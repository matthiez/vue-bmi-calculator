# BmiCalculator

**USAGE**

Import the default export and render the component.

```vue
    <template>
            <bmi-calculator></bmi-calculator>
    </template>
    <script>
       import BmiCalculator from "vue-bmi-calculator"
   
       export default {components: {BmiCalculator}}
    </script>
```
